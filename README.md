# STS Practice Hub

A minimal, high-performance web application for practicing STS questions.

## Features
- **Read Mode**: Browse all questions and explanations.
- **Quiz Mode**: Practice with instant feedback.
- **Test Mode**: Simulate a real exam environment.
- **Dark/Light Mode**: Optimized for comfortable reading.

## Structure
- `Style/`: Contains CSS files.
- `Script/`: Contains JavaScript logic.
- `Json/`: Contains quiz data.
- `index.html`: Main entry point.

## How to Run
Simply open `index.html` in any modern web browser.
